## 🚀 Мои проекты
# Task Manager
Полноценное веб-приложение для Windows по управлению задачами. Мой самый проработанный проект, где я применил весь основной стек: Spring Boot, Security, Hibernate, PostgreSQL. Умеет всё, что нужно для учета задач в команде или личного использования.
# Employee Audit Service
Консольное приложение для управления сотрудниками с разграничением прав доступа (админ, HR, менеджер, пользователь). Ключевая особенность — реализация валидации и логирования через Spring AOP + паттерн Command. Показывает, как можно строить гибкую архитектуру без захламления бизнес-логики.
# Course Management System
Консольное приложение для управления учебным процессом (факультеты, преподаватели, студенты, курсы). Удобный интерфейс, гибкие запросы к данным, управление связями между сущностями.
